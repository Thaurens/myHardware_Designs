## 🌟 Project Highlights

* **Ultra-Minimalist Design:** The entire circuit utilizes only four easily identifiable, through-hole components, making the BOM and sourcing incredibly manageable.
* **Incredibly Simple Assembly:** No microscopic surface-mount (SMD) parts. Ideal for learning how to solder with immediate visual results.
* **Core Electronic Principles:** Demonstrates the most fundamental hardware loop: a power source, a mechanical switch, a current-limiting resistor, and an LED.
* **Efficient Board Layout:** Intuitive component placement on a highly efficient board, minimizing manufacturing complexity.

## ⚙️ Technical Specifications
<div align="center">

| Component | Details / Value |
| :--- | :--- |
| **Power Source (BT1)** | 3V CR2032 Coin Cell Battery |
| **Light Source (D1)** | Standard 5mm Through-Hole LED |
| **Control (SW1)** | 4-Pin Tactile Push Button |
| **Current Limiter (R1)** | Axial Through-Hole Resistor |
